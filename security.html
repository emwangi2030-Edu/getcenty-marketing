Security

Centy is built for regulated, high-trust workflows — payments, payroll, and employee data. We treat security and data protection as ongoing priorities, not a one-time checklist.

What we focus on

  • Encrypted transport (HTTPS) for the marketing site and application.
  • Role-based access and audit-friendly workflows inside the product.
  • Vendor and infrastructure practices aligned with common enterprise review themes.

Responsible disclosure

If you believe you have found a security issue, please email hello@centycapital.com with a clear description and steps to reproduce. We appreciate coordinated disclosure.

Questions

For security questionnaires, DPIAs, or vendor assessments, contact hello@centycapital.com.
