# Import these files into Google Drive (emwangi2030@gmail.com)

These CSVs are formatted for **Google Sheets** (one tab per file). This environment **cannot** log in to your Google account—**you** upload or import them in ~2 minutes.

## Option A — New workbook (recommended)

1. Sign in to [Google Drive](https://drive.google.com) as **emwangi2030@gmail.com**.
2. **New → Google Sheets → Blank spreadsheet**.
3. Name it: **`Centy Kenya SEO — 90-Day Command Center (2026)`**.
4. Rename **Sheet1** to **`Content_Calendar`**.
5. **File → Import → Upload** → select `01_Content_Calendar_90d.csv` → **Import location:** *Replace current sheet* → Import.
6. Click **+** (add sheet), name **`Weekly_Metrics`**, then **File → Import** `02_Weekly_Metrics_Tracker.csv` → **Insert new sheet(s)**.
7. Repeat for:
   - `03_North_Star_Targets.csv` → tab **`North_Star`**
   - `04_Competitor_Watchlist.csv` → tab **`Competitors`**
   - `05_Keyword_Map_Base.csv` → tab **`Keyword_Map`**
8. **Move** the file into whatever Drive folder you use (e.g. `Centy / Marketing / SEO`).

## Option B — Zip and upload, then import inside Sheets

1. Zip this folder locally (if you sync the repo):  
   `docs/google-drive-export/*.csv`
2. Upload the zip to Drive (optional), unzip in Drive or on your machine.
3. Import each CSV into the workbook as in Option A.

## After import

- **Share** with your content lead + compliance reviewer (Viewer or Commenter as appropriate).
- **Lock** header rows: View → Freeze → 1 row.
- Fill **`live_url`**, **`owner`**, **`status`** in the calendar as posts ship.
- Paste GSC totals into **`Weekly_Metrics`** every Monday.
